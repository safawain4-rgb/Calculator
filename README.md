# Calculator
# 🐻🌸 Kawaii Calculator

A cute, animated, bear-themed calculator built with **HTML, CSS, and JavaScript**. Designed with pastel aesthetics, smooth animations, interactive sound effects, theme switching, and a delightful user experience.

![Kawaii Calculator](https://img.shields.io/badge/Theme-Kawaii%20UI-pink)
![HTML](https://img.shields.io/badge/HTML-5-orange)
![CSS](https://img.shields.io/badge/CSS-3-blue)
![JavaScript](https://img.shields.io/badge/JavaScript-ES6-yellow)

---

## ✨ Features

### 🧮 Calculator Functions

* Basic arithmetic operations

  * Addition (+)
  * Subtraction (−)
  * Multiplication (×)
  * Division (÷)
* Percentage calculations (%)
* Clear All (AC)
* Memory functions:

  * M+
  * M−
  * MRC (Memory Recall / Clear)

### 🎨 Cute User Interface

* Kawaii bear-inspired design
* Soft pastel color palette
* Animated splash screen
* Floating particles and sparkles
* Responsive mobile-first layout
* Smooth button animations

### 🐾 Theme Switching

Switch between two adorable themes:

#### 🌸 Pink & Pastel Theme

* Soft pink background
* Cute kawaii aesthetic
* Gentle pastel colors

#### 🐻 Brown & Yellow Theme

* Cozy bear-inspired colors
* Warm brown tones
* Golden yellow accents

### 🔊 Sound Effects

* Interactive button tap sounds
* Theme-switching sounds
* Toggle sound ON/OFF in settings

### ⚙️ Settings Panel

* Change calculator themes
* Enable/disable sound effects
* Accessible keyboard navigation

### 📱 Mobile Friendly

* Fully responsive design
* Touch-optimized controls
* Mobile viewport support
* Haptic vibration feedback (supported devices)

## 🛠 Installation

1. Clone the repository:

```bash
git clone https://github.com/your-username/kawaii-calculator.git
```

2. Open the project folder:

```bash
cd kawaii-calculator
```

3. Run the project:

Simply open `index.html` in your browser.
